# Password Saver - Complete Feature Guide

## 📱 App Overview
Password Saver is a secure, user-friendly mobile application that helps you store, manage, and organize all your passwords in one safe place. With encrypted storage, biometric authentication, and powerful search features, managing passwords has never been easier.

---

## 🔐 Security Features

### App Security & Authentication
- **Enable/Disable Security**: Control whether the app requires authentication when opening
- **Biometric & PIN Authentication**: Use your device's fingerprint, face recognition, or PIN for quick access
- **Device Lock Support**: Works with PIN, pattern, password, and biometric methods
- **Encrypted Storage**: All passwords are encrypted and stored locally on your device
- **No Cloud Sync**: Your data stays on your device - complete privacy

---

## 💾 Password Management

### Add Passwords
- **One-Tap Add**: Press the floating button to add a new password entry
- **Required Fields**: Title, Password, and Email address (validated for correct format)
- **Optional Fields**: Username (if different from email)
- **Category Selection**: Organize passwords by categories (e.g., Social Media, Work, Banking)
- **Edit Reason Tracking**: When editing, note why you changed the password for future reference

### Edit Passwords
- **Edit Existing Entries**: Tap any password card to open detailed view, then edit
- **Change History**: View when and why each password was last modified
- **Email Validation**: New email addresses are validated before saving
- **Password Strength Indicator**: See visual feedback on password security level

### View Password Details
- **Flip Card Display**: Tap passwords in the list to flip and reveal them
- **Full Details Page**: See all information including email, username, and modification history
- **Copy to Clipboard**: One-tap copy for password, email, and username
- **Visibility Toggle**: Show/hide passwords for privacy when viewing in detail
- **Edit & Delete**: Quick access to edit or remove entries

### Delete Passwords
- **Individual Delete**: Remove single passwords with confirmation
- **Bulk Delete**: Select multiple passwords and delete them together
- **Confirmation Dialog**: Prevents accidental deletion with a confirmation prompt

## 🎯 Workflow Examples

### Adding Your First Password
1. Tap the floating **+** button at the bottom-right
2. Enter password title (e.g., "Gmail")
3. Enter your password (will show strength level)
4. Enter email address (must be valid format)
5. Optionally enter username if different
6. Select a category or create new one
7. Tap Save

### Editing an Existing Password
1. Tap the password card to open details
2. Tap the Edit pencil icon
3. Update any field
4. Enter a reason for the change (e.g., "Updated for security")
5. Tap Save

### Finding a Password
1. Tap the search icon in the top bar
2. Type any part of what you're looking for: title, email, or username
3. Results filter in real-time
4. Tap on a result to view full details

### Bulk Deleting Passwords
1. Long-press any password entry to enter multi-select mode
2. Tap checkboxes to select multiple passwords
3. All selected passwords appear in the top bar
4. Tap the delete icon in the selection bar
5. Confirm the deletion

## 📞 Troubleshooting

### FAB Button Overlapping Passwords
- **Solution**: Go to menu → Security Settings → Change "Add Button Position" to show in its own space

### Can't Find a Password
- **Solution**: Use the search feature to look by title, email, or username

### Forgot an Email Format
- **Solution**: When adding/editing, the app validates email format automatically

### Lost Access to App
- **Solution**: If security is disabled in settings, app opens normally. If enabled, use your device's biometric or PIN

### Want to Disable Authentication
- **Solution**: Go to menu → Security Settings → Toggle "Enable App Security" off
